macro_rules! parse_result {
    ($output:expr, $err_statement:expr) => { ... };
}

This macro is intended to be used in all cases where we get an Err out of Result and want to package it into safe_ffi::errors::FfiError::SpecificParseError(String). This is useful because there may be miscellaneous erros while parsing through a valid JSON due to JSON not conforming to certain mandatory requirements. This can then be communicated back to the JSON sending client.

Examples

#[derive(Debug)]
enum SomeSpecialError {
    Zero,
    One,
}

fn f() -> Result<String, SomeSpecialError> {
    Err(SomeSpecialError::One)
}

fn g() -> Result<(), safe_ffi::errors::FfiError> {
    let _module = try!(parse_result!(f(), ""));

    Ok(())
}

fn main() {
    if let Err(err) = g() {
        println!("{:?}", err);
    }
}
